El módulo Tkinter (“interfaz Tk”) es la interfaz estándar de Python para el kit de herramientas Tk GUI de Scriptics (anteriormente desarrollado por Sun Labs). Tanto Tk como Tkinter están disponibles en la mayoría de las plataformas Unix, así como en los sistemas Windows y Macintosh. A partir de la versión 8.0, Tk ofrece un aspecto nativo en todas las plataformas. Tkinter consta de una serie de módulos. La interfaz Tk es proporcionada por un módulo de extensión binario llamado _tkinter . Este módulo contiene la interfaz de bajo nivel para Tk, y nunca debe ser utilizado directamente por los programadores de aplicaciones. Por lo general, es una biblioteca compartida (o DLL), pero en algunos casos puede estar vinculada estáticamente con el intérprete de Python. La interfaz pública se proporciona a través de varios módulos de Python. El módulo de interfaz más importante es el propio módulo Tkinter .
